		API01: お問い合わせ一覧取得						
			項目	内容				
			エンドポイント	/api/contacts				
			メソッド	GET				
			認証	不要				
			認可（ポリシー）	-				
			説明	お問い合わせ一覧を取得する。検索・ページネーション対応				
								
			リクエストパラメータ					
			パラメータ名	型	必須	バリデーション	説明	例
			keyword	string	任意	最大255文字	氏名またはメールアドレスで部分一致検索	田中
			gender	integer	任意	1, 2, 3のいずれか	性別で絞り込み（1:男性, 2:女性, 3:その他）	1
			category_id	integer	任意	categoriesテーブルに存在するID	お問い合わせ分類で絞り込み	1
			date	string	任意	YYYY-MM-DD形式	作成日で絞り込み	2024-01-15
			page	integer	任意	1以上の整数（デフォルト: 1）	ページ番号	1
			per_page	integer	任意	1〜100（デフォルト: 7）	1ページあたりの件数	7
								
			レスポンス					
			ステータス	説明	レスポンスボディ例			
			200 OK	取得成功	{ "data": [...], "meta": { "current_page": 1, ... } }			
								
								
		API02: お問い合わせ新規作成						
			項目	内容				
			エンドポイント	/api/contacts				
			メソッド	POST				
			認証	不要				
			認可（ポリシー）	-				
			説明	新しいお問い合わせを作成する				
								
			リクエストパラメータ					
			パラメータ名	型	必須	バリデーション	説明	例
			category_id	integer	必須	categoriesテーブルに存在するID	お問い合わせ分類のID	1
			first_name	string	必須	最大255文字	姓	田中
			last_name	string	必須	最大255文字	名	太郎
			gender	integer	必須	1, 2, 3のいずれか	性別（1:男性, 2:女性, 3:その他）	1
			email	string	必須	メール形式、最大255文字	メールアドレス	tanaka@example.com
			tel	string	必須	最大5桁×3グループ（合計10〜11桁、ハイフンなし）	電話番号	09012345678
			address	string	必須	最大255文字	住所	東京都渋谷区1-1-1
			building	string	任意	最大255文字	建物名	〇〇マンション101
			detail	string	必須	最大120文字	お問い合わせ内容	商品が届きません
			tags	array	任意【応用】	tagsテーブルに存在するIDの配列	タグIDの配列	[1, 2]
								
			レスポンス					
			ステータス	説明	レスポンスボディ例			
			201 Created	作成成功	{ "data": { "id": 1, ... } }			
			422 Unprocessable Entity	バリデーションエラー	{ "message": "...", "errors": {...} }			
								
								
		API03: お問い合わせ詳細取得						
			項目	内容				
			エンドポイント	/api/contacts/{id}				
			メソッド	GET				
			認証	不要				
			認可（ポリシー）	-				
			説明	指定したIDのお問い合わせ詳細を取得する				
								
			リクエストパラメータ					
			パラメータ名	型	必須	バリデーション	説明	例
			id	integer	必須（URL）	-	お問い合わせID	1
								
			レスポンス					
			ステータス	説明	レスポンスボディ例			
			200 OK	取得成功	{ "data": { "id": 1, "category": {...}, ... } }			
			404 Not Found	データが存在しない	{ "message": "お問い合わせが見つかりません" }			
								
								
		API04: お問い合わせ削除						
			項目	内容				
			エンドポイント	/api/contacts/{id}				
			メソッド	DELETE				
			認証	不要				
			認可（ポリシー）	-				
			説明	指定したIDのお問い合わせを削除する				
								
			リクエストパラメータ					
			パラメータ名	型	必須	バリデーション	説明	例
			id	integer	必須（URL）	-	お問い合わせID	1
								
			レスポンス					
			ステータス	説明	レスポンスボディ例			
			204 No Content	削除成功	（レスポンスボディなし）			
			404 Not Found	データが存在しない	{ "message": "お問い合わせが見つかりません" }			
			403 Forbidden	【応用】認可エラー	{ "message": "この操作を行う権限がありません" }
